# Assets Folder

This folder contains images, icons, and other media files for the SouliTEK All-In-One Scripts project.

## Folder Structure

### `/images`
General images, logos, banners, and graphics used in the project.
- Project logos
- Header images
- Promotional graphics
- Documentation images

### `/icons`
Icon files for the launcher and tools.
- Tool icons (.ico, .png)
- Application icons
- UI elements

### `/screenshots`
Screenshots of the tools and launcher in action.
- Launcher screenshots
- Tool demonstrations
- Before/after examples
- Feature highlights

## File Formats

Supported file formats:
- **Images**: PNG, JPG, JPEG, GIF, BMP
- **Icons**: ICO, PNG (for launcher icons)
- **Vector**: SVG (if needed)

## Usage

1. Upload files to the appropriate subfolder
2. Reference them in documentation using relative paths
3. Keep filenames descriptive and lowercase with hyphens (e.g., `launcher-main-window.png`)

## Notes

- Keep file sizes reasonable (compress images when possible)
- Use descriptive filenames
- Maintain organized subfolders
- Update this README if adding new categories
